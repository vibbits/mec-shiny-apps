What does the app do?

VIB Metabolomics Core - Leuven

BNX_viewer
The BNX_viewer app loads BNX data from a file and creates scatterplots or histograms with filtering (no export at this point, one can easily use built-in functions for that purpose). A sample BNX file can be downloaded from the App for testing.
